# Real-Time Multi-Object Detection HUD (AI Vision System)

This project is a real-time AI vision system that uses OpenCV and YOLOv8 to detect multiple objects from your webcam feed. It overlays a clean, futuristic HUD (Heads-Up Display) showing bounding boxes, class labels, confidence scores, FPS, and more—just like a modern AI vision system.

## Features
- Real-time object detection using YOLOv8 (Ultralytics)
- Webcam video feed with HUD overlay
- Bounding boxes, class labels, and confidence scores
- FPS and inference time display
- Smooth, minimal, and futuristic UI

   ```

   ```
   pip install opencv-python ultralytics numpy
   ```
4. Run the project:
   ```
   python main.py
